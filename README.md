# 🌍 Global Airbnb Performance Dashboard

A comprehensive Power BI dashboard analyzing Airbnb listings, host performance, customer ratings, pricing trends, and review behavior across major global cities.

---

# 📌 Project Overview

This project was developed to analyze Airbnb performance using Power BI. The dashboard provides business insights into listing growth, pricing, customer ratings, review frequency, seasonality, and host trust indicators.

The objective was to transform raw Airbnb data into an interactive business intelligence solution that supports data-driven decision making.

# 📊 Dashboard Pages

### 1. Overview
Provides a high-level summary of Airbnb listings, cities, hosts, property types, review counts, and listing growth trends.

### 2. Ratings Analysis
Analyzes customer ratings, average prices, city comparisons, and overall market performance.

### 3. Reviews Analysis
Explores customer review frequency, seasonality, trust indicators, and review behavior.
